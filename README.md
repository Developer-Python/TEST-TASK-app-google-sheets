# currency-handler
Для правельности выполнение программы требуется следущие pip пакеты:

1) cachetools == 3.1.1
2) google-api-python-client == 1.7.11
3) google-auth == 1.6.3
4) google-auth-httplib2 == 0.0.3
5) httplib2 == 0.13.1
6) oauth2client == 4.1.3
7) pyasn1 == 0.4.6
8) pyasn1-modules == 0.2.6
9) rsa == 4.0
10) six == 1.12.0
11) uritemplate == 3.0.0
